# **App Name**: HUSTLERSPOINT

## Core Features:

- Community BLOGS: Displays a real-time feed of posts and updates from community members.
- Group DISCUSSION FORAMS: Allows users to create and join groups based on shared interests.
- Member Profiles: Enables users to create and customize their profiles, showcasing their skills and interests.
- Curated News: Provides a daily digest of relevant news articles and industry updates from linkdin producthunt and reddit .
- Newsletter Integration: Allows users to subscribe to a newsletter for exclusive content and updates.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) to convey trust and professionalism.
- Background color: Light gray (#ECEFF1), offering a clean, modern look.
- Accent color: Soft orange (#FFAB40) for CTAs and highlights.
- Headline font: 'Poppins', sans-serif for a contemporary, precise feel.
- Body font: 'PT Sans', sans-serif, pairing with Poppins, used to maintain readability.
- Use modern, minimalist icons for navigation and key features.
- Follow the structure and responsive behavior of the getmaterial repository. Key routes: /, /feed, /groups/:id, /members/:username
